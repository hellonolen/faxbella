Section 1

The Vision

FaxBella is not a fax app with AI bolted on. FaxBella is an AI-powered document routing platform that happens to support fax as one delivery channel among many.

Gemini is the brain. Everything else is plumbing. Documents come in any way. AI reads, understands, classifies, and routes them. Delivery happens through whatever channel the recipient needs — fax, email, secure link, webhook, or API.

Who Uses FaxBella

User What They Do How They Access It
Regular person Opens the app, uploads a document, sends a fax Day Pass ($9) or Dashboard at faxbella.com
Title company Sends closing packages to attorneys and lenders Dashboard + bulk upload
Pharmacy / clinic Receives prescriptions and referrals, AI routes to correct department Dashboard + email intake
Insurance company Processes claims documents from multiple sources API + upload portal
Car dealership Receives finance docs, titles, trade paperwork Dashboard + email intake
Developer Integrates document routing into their own app MCP server + REST API
AI agent Automated workflows that send and route documents MCP tool calls

Same platform. Same AI engine. Four doors in: Dashboard for regular people. Upload portal for senders. MCP for developers. API for automation.

Section 3

AI Agents

FaxBella is a fully agentic environment. Five AI agents handle the complete document lifecycle.

Agent Role How It Works
Intake Agent Accepts documents from any source Upload portal, email, API, MCP tool call, drag-and-drop. Gemini instantly reads and understands the document the moment it arrives.
Routing Agent Decides where it goes Gemini analyzes content, identifies the recipient, matches to known contacts, learns routing patterns over time. Gets smarter with every document.
Delivery Agent Sends it however the recipient needs Fax (via FreeSWITCH), email (via EmailIt), secure download link, API webhook. Recipient preference drives channel selection.
Follow-up Agent Confirms delivery and handles failures Tracks delivery confirmations. Retries on failure. Escalates when needed. Notifies the sender of final status.
Concierge Agent Talks to the user Chat interface in the dashboard + voice via Gemini. "Send this to Dr. Smith" and the agent handles everything. Proactive: "I noticed 3 urgent referrals — want me to route them?"
Section 4

MCP Server — The Developer Revenue Channel

The MCP (Model Context Protocol) server exposes FaxBella's capabilities as tools that other AI systems can use. This is not just a technical feature — it is a distribution and revenue strategy.

Developers discover FaxBella's MCP server. They connect it to their AI workflows. Every document processed through their integration means FaxBella gets paid. They tell other developers. It shows up in MCP registries. Organic growth without marketing spend.

MCP Tools Exposed

Tool What It Does Example Use

Section 5

Infrastructure Choices

Fax Transmission: FreeSWITCH (Open Source, Self-Hosted)

FreeSWITCH is the industry-standard open-source telecom engine. It handles the fax protocol — converting PDFs into fax signals and vice versa. We deploy it on our own server and own it completely.

Component What It Is Why
FreeSWITCH Open-source telecom engine (3,000+ GitHub stars) Battle-tested, handles T.38 fax protocol, REST API, massive community
mod_spandsp FreeSWITCH module for fax Converts PDF to fax signal and back. The actual engine that makes fax work.
SpanDSP DSP library for fax/modem signals Core dependency. Handles the low-level signal processing.

SIP Trunk Provider: VoIP.ms

A SIP trunk connects our FreeSWITCH server to the phone network. VoIP.ms is a Canadian company with minimal signup requirements and pay-as-you-go pricing.

Section 8

Multi-Channel Document Intake

Documents arrive from five different channels. All of them feed into the same Gemini AI pipeline.

Channel How It Works Who Uses It
Upload Portal Public page per customer. Drag-and-drop any document. Gemini reads it instantly. Anyone — regular users, offices, walk-ins
Email Intake Cloudflare Email Workers receive documents at customer@intake.faxbella.com. Attachments extracted automatically. Offices forwarding from existing fax-to-email, printers doing scan-to-email
Fax Receive FreeSWITCH receives fax calls on dedicated numbers. Converts to PDF. Feeds into AI. Anyone sending a traditional fax
REST API POST a document with metadata. Returns classification, routing, and delivery status. Developers integrating FaxBella into their own systems
MCP Server AI agents call faxbella.sendDocument() or faxbella.routeDocument() as tool calls. AI workflows, automated systems, developer agents
Section 9

Multi-Channel Delivery

Gemini's Routing Agent picks the best delivery channel based on the recipient's preferences and the document type.

Channel How It Works When It's Used
Fax FreeSWITCH converts PDF to fax signal, sends via SIP trunk to recipient's fax machine Recipient only has a fax number (pharmacies, courts, older offices)
Email EmailIt delivers the document as a PDF attachment with AI-generated summary Recipient prefers email (most modern offices)
Secure Link Document stored in R2, time-limited presigned URL sent to recipient Large files, HIPAA-sensitive documents, one-time access needed
Webhook POST to customer's configured endpoint with document data and metadata Automated systems, EHR integrations, custom workflows
API Response Return document data directly in the API/MCP response Developer integrations, AI agent workflows
